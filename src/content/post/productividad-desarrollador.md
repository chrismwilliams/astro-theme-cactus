## Optimizando el día a día

Como desarrolladores, pasamos muchas horas frente al ordenador. Usar las herramientas correctas puede marcar una gran diferencia en nuestra productividad y bienestar.

### 1. VS Code con las extensiones correctas

Visual Studio Code es mi editor de código principal. Estas son las extensiones que no pueden faltar:

- **GitHub Copilot**: Autocompletado inteligente con IA
- **Error Lens**: Muestra los errores directamente en la línea
- **GitLens**: Información de Git integrada en el editor

### 2. Warp Terminal

Un terminal moderno con autocompletado inteligente y una interfaz mucho más amigable que las opciones tradicionales.

### 3. Raycast (Mac) / PowerToys (Windows)

Lanzadores de aplicaciones que me permiten hacer casi todo sin tocar el ratón:

- Abrir aplicaciones
- Buscar archivos
- Ejecutar scripts personalizados
- Gestionar ventanas

### 4. Obsidian para notas

Todas mis notas, ideas y documentación personal están en Obsidian. El sistema de enlaces bidireccionales es perfecto para organizar el conocimiento.

### 5. Fig para el terminal

Autocompletado visual para el terminal que muestra sugerencias mientras escribes comandos.
